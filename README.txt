Topics
------

1. How to install?
------------------

- This application has been developed for linux environment.
- To install familiarityByClass application:
    
    make



2. How to run?
--------------

- To run this application:
   ./familiarityByClass sequence.fasta familyDatasetFolder outputFolder


- The result of this application is at outputFolder/ folder
